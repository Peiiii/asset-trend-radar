# 概览筛选与机会事件聚合验证报告

## 范围

- 概览页继续保留市场、品种、层级、主题、信号、排序、时间范围、周期、搜索、快捷策略和当前筛选 chip。
- 概览页不展示图表墙专属的卡片/表格视图切换。
- 机会事件按当前筛选后的可见资产聚合，避免同一资产多条事件刷屏。
- 机会事件卡可点击进入资产详情，并可通过详情页左上返回按钮回到原筛选口径。
